Unduh Laporan Progres & Log Kendala (Format Excel)

Unduh File Laporan Kerja Lapangan

Klik tombol di bawah ini untuk mengekstrak data dari database secara terstruktur. File Excel yang diunduh mencakup ringkasan status sub-SLS per Kecamatan/Desa serta catatan lengkap log kendala petugas.

File Excel yang akan diunduh berisi:

Sheet 1: Summary Progres Lapangan

Berisi daftar seluruh Sub-SLS wilayah tugas dengan data target muatan, jumlah assignment (Open, Submit, Reject, Pending, Approved), persentase progres, dan nama petugas penanggung jawab.

Sheet 2: Log Kendala Lapangan

Berisi daftar Sub-SLS yang dilaporkan bermasalah atau tidak selesai dengan status 'tidak_selesai_kendala', lengkap dengan deskripsi kendala yang diisi petugas di lapangan.
